# Сайт Ярослава

Личная страница-визитка + раздел для дел: контент-план, финансы, фитнес, соцсети, проекты.
Статический сайт (HTML/CSS/JS), без сервера — идеально для бесплатного хостинга на GitHub Pages.

## Файлы

- `index.html` — структура страницы
- `style.css` — оформление
- `script.js` — рендер данных, тикер, кнопка быстрого добавления ссылки
- `data.js` — **всё содержимое сайта**. Редактируй только этот файл, чтобы менять разделы и ссылки.

## Как опубликовать на GitHub Pages (без установки чего-либо)

1. Зайди на [github.com](https://github.com) и войди (или зарегистрируйся — бесплатно).
2. Нажми **New repository** (зелёная кнопка).
   - Имя репозитория: например `my-site` (можно любое).
   - Сделай его **Public**.
   - Не добавляй README при создании — просто "Create repository".
3. На странице пустого репозитория нажми **uploading an existing file**.
4. Перетащи туда все файлы из этой папки: `index.html`, `style.css`, `script.js`, `data.js`, `.nojekyll`, `README.md`.
5. Нажми **Commit changes**.
6. Зайди в **Settings → Pages** (в меню слева).
7. В разделе **Build and deployment → Source** выбери **Deploy from a branch**.
8. Branch: `main`, папка `/ (root)` → **Save**.
9. Подожди 1–2 минуты, обнови страницу — вверху появится ссылка вида:
   `https://твой-логин.github.io/my-site/`

Это и есть твой сайт, бесплатно и навсегда (пока репозиторий существует).

## Как редактировать контент дальше

Открой `data.js` прямо на GitHub (иконка карандаша ✏️ в правом верхнем углу файла),
добавь/измени ссылки по образцу внутри файла, нажми **Commit changes** — сайт обновится
за 1–2 минуты сам.

## Кнопка "+ ссылка" на самом сайте

Позволяет быстро добавить ссылку в любой раздел прямо из браузера — но такая ссылка
сохраняется **только в этом браузере** (через localStorage), не на GitHub. Для того чтобы
ссылка была видна всем и навсегда — добавляй её в `data.js` и публикуй заново.
